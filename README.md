# Titanic-Dataset
Exploratory Data Analysis of Titanic Dataset in Python. 

- Normalized and pre-processed the Titanic dataset in Excel to handle inconsistencies before importing it into Python using Pandas.
- Performed comprehensive data cleaning to address missing values, duplicates, and outliers, ensuring data integrity.
- Conducted Exploratory Data Analysis (EDA) to identify trends in survival rates across genders and analyze the impact of age on survival.
- Applied descriptive statistics and visualized key insights through bar charts, histograms, and box plots using Matplotlib and Seaborn.
- Derived actionable insights highlighting the influence of gender, age, and class on survival outcomes.
- Strengthened Python skills by working with dataframes, filtering, and groupby functions to extract meaningful insights from raw data.
